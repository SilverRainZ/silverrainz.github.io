====
摸底
====

画室报到第一天，晓飞老师要求独立写生一张石膏像， 我选了当时离着最近的米开朗基罗像。

其实这不是个好选择，这个石膏被放在地上，同时受两个顶灯的照射，光线很散，实在不太好画。

出现的问题：

- 阴影的形状没有和大的形体匹配

    - 比如上身的布袍的塑造没有体现出胸腔的体积

- 各个灰度的色层没有区分开来，意识上和手上都没有，导致无法继续塑造
